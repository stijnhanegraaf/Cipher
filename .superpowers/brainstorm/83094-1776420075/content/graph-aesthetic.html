Graph aesthetic — pick a direction

Current graph = basic grayscale dots. Which visual soul should we chase for "insane good POW POW"?

A · Constellation

Constellation

Dark cosmic bg with subtle nebula mist. Tiny star-like dots with soft bloom on hubs. Edges are whisper-thin rays that light up on hover. Moody, quiet, almost meditative. Best for big vaults — the density feels like a sky, not a mess.

B · Linear clean

Linear clean

Flat dark bg. Nodes colored by folder (indigo / green / amber / gray) — you read clusters immediately. Solid circles, crisp edges, generous hover halo, selection ring in brand indigo. Feels like a Linear dashboard chart, not a vault viz. Most on-brand.

C · Schematic

Schematic

Light-mode engineering diagram vibe. Grid background, hard geometry, small inline text labels on clusters. Feels like a design doc or architecture map. Maximum clarity, less mood.
